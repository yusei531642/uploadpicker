# UploadPicker

UploadPicker is a local-first hybrid image search tool for searching uploaded photos by natural-language traits.

## Windows app-like usage

- Double-click `UploadPicker.bat` to open the setup wizard.
- Click `Install / Repair` to install Python if needed, create `.venv`, install dependencies, and create shortcuts.
- Click `Launch App` to start the local server and open the browser.
- Click `Uninstall` or run `Uninstall UploadPicker.bat` to remove `.venv`, local data, and shortcuts.

The wizard is designed so that even users without Python can get started on Windows, as long as `winget` is available.

## Install-only EXE

- Build the installer EXE with `Build Install EXE.bat` or `installer/build_install_exe.ps1`.
- The built file will be generated by PyInstaller under `dist/UploadPicker-Install.exe`.
- Distribute that EXE together with the project folder so it can create `.venv`, install dependencies, and set up shortcuts.
- This EXE is for `Install / Repair` only. Launch and uninstall continue to use the generated shortcuts and existing scripts.

## Current scope

- FastAPI app with local upload flow
- Search UI and API skeleton
- GPU-ready architecture for hybrid search
- Storage layout for images, thumbnails, metadata, and embeddings

## Planned search pipeline

1. Save uploaded images locally
2. Generate thumbnails
3. Build image embeddings with a CLIP-family model
4. Add person and face detection
5. Filter by attributes and rerank results

## Run

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -e .
uvicorn app.main:app --reload
```

Open `http://127.0.0.1:8000`.

## Developer launch shortcut

After installation, you can also launch with:

```bash
uploadpicker
```

## Project layout

- `app/main.py` FastAPI entry point
- `app/runner.py` local launcher entry point
- `app/config.py` runtime settings
- `app/db.py` database setup
- `app/models.py` SQLModel tables
- `app/services/` indexing and search services
- `app/templates/` server-rendered UI
- `data/` local storage for images and indexes
- `installer/` Windows setup and launch scripts
